# Interactive Data Dashboard

Interactive Data Dashboard — это веб-приложение для анализа CSV и JSON данных прямо в браузере.  
Пользователь может загрузить файл, просматривать таблицу, применять фильтры, сортировку, получать статистику и строить графики.

🔗 **Деплой:**  
https://interactive-data-dashboard.vercel.app/

---

# Возможности

- Загрузка **CSV и JSON** файлов
- **Drag & Drop** загрузка
- Табличное отображение данных
- **Поиск по всем колонкам**
- **Фильтрация по колонке**
- **Сортировка**
- **Пагинация**
- **Статистика по числовым колонкам**
  - count
  - sum
  - average
  - min / max
- **Интерактивный график**
- Demo dataset
- Полностью работает **на клиенте (без backend)**
